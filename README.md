# WakeMyWacom

WakeMyWacom is a tiny macOS utility that fixes a common Wacom issue with one click.

If your Wacom tablet stops responding after sleep, it’s usually not the tablet itself. One or more Wacom background services get stuck. Restarting those services fixes the problem, but doing it manually or rebooting your Mac breaks your flow.

WakeMyWacom restarts the stuck services and quits immediately. Nothing runs in the background.

This repository contains the core script logic for transparency.

You can download the packaged macOS app here:  
👉 https://sillysoft.gumroad.com/l/WakeMyWacom

WakeMyWacom does not collect data, does not run continuously, and does not phone home. All state is stored locally on your machine.
